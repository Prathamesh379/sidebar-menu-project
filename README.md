# sidebar-menu-project
A responsive sidebar menu built using HTML and CSS with smooth transitions and modern UI design.

## Features
- Responsive sidebar layout
- Smooth open/close animation
- Font Awesome icons
- Clean and modern UI
- Pure HTML and CSS (no JavaScript)

## Technologies Used
- HTML5
- CSS3
- Google Fonts (Poppins)
- Font Awesome Icons

## Project Structure

sidebar-menu-project/
│
├── index.html
├── style.css
└── photo.jpg


## How to Run
1. Download or clone the repository:
2. Open the folder.
3. Double-click `index.html` to open it in your browser.

## Preview
The sidebar menu appears on the left side and can be toggled using the menu icon.
